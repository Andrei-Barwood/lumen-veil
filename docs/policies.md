# Sorox and Vossk Policies

## Sorox

- sanctity of corridor geometry,
- little patience for ambiguity,
- swift denial where grace is unearned,
- early containment at sanctuary breach,
- ceremonial ascent for vessels that remain in accord.

Sorox therefore tends to:

- measure,
- know,
- bless,
- release,

yet will pass directly into containment when sanctity is crossed.

## Vossk

- judgment led by bearing and pattern,
- measured tolerance for incomplete identity,
- escalation shaped by conduct rather than title alone,
- preference for shadow and degrade before closure of the field.

Vossk therefore tends to:

- observe unknown traffic longer,
- shadow suspicious drift,
- degrade persistent anomalies,
- contain only when the pattern refuses discipline.

## Rule Format

Each jurisdiction file contains an ordered list of rules with:

- `priority`
- `match`
- `action`
- `target_state`
- `reason_code`
- optional `effects`
- optional emitted events
