# Glossary

- `Lumen Veil`: the platform as a whole.
- `Witness Array`: abstract sensing cluster for remote signatures and approach patterns.
- `Seal`: the transponder identity token.
- `Canon`: the declarative policy engine.
- `Mercy`: the reversible containment layer.
- `Threshold`: monitored jurisdictional ingress radius.
- `Sanctuary`: restricted sacred zone.
- `Ward Node`: distributed field emitter in the perimeter lattice.
- `Transit Purified`: successful release after judgment.
- `Silence Invoked`: abstract communications and subsystem suppression event.
